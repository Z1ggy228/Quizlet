# Слова — личный тренажёр английского

Аналог Quizlet для одного пользователя: папки → наборы → карточки, режимы Flashcards и Learn,
хранение в Supabase (данные и картинки доступны с любого устройства под одним аккаунтом).

> Здесь — как запустить. Почему всё устроено именно так и на какие грабли уже наступили —
> в [CONTEXT.md](CONTEXT.md). Читать перед тем, как что-то менять.

## Запуск

1. **Создайте проект** на [supabase.com](https://supabase.com) (бесплатного тарифа хватает).

2. **Примените миграции** по порядку. В панели Supabase: `SQL Editor` → `New query` →
   вставьте файл целиком → `Run`.

   1. [`supabase/migration.sql`](supabase/migration.sql) — таблицы `folders`, `sets`,
      `cards`, политики RLS и бакет `card-images`.
   2. [`supabase/migration_02_positions.sql`](supabase/migration_02_positions.sql) —
      колонки `position` для явного порядка наборов и карточек.
   3. [`supabase/migration_03_srs.sql`](supabase/migration_03_srs.sql) — поля
      интервального повторения и словаря, таблицы `user_settings` и `study_days`.
   4. [`supabase/migration_04_day_goal.sql`](supabase/migration_04_day_goal.sql) — дневная
      цель запоминается в самом дне, чтобы её изменение не переписывало историю.

   Все скрипты идемпотентные, повторный запуск ничего не сломает.

3. **Вставьте ключи.** Откройте [`src/lib/supabase.js`](src/lib/supabase.js) и замените
   значения двух переменных вверху файла:

   ```js
   export const SUPABASE_URL = 'https://ваш-проект.supabase.co'
   export const SUPABASE_ANON_KEY = 'eyJhbGciOi...'
   ```

   Взять их можно в панели Supabase: `Settings` → `API`.
   `SUPABASE_URL` — это поле **Project URL**, `SUPABASE_ANON_KEY` — ключ **anon / public**.
   Ключ `service_role` сюда вставлять нельзя.

4. **Ключ для подбора картинок** (необязательно). Откройте
   [`src/lib/images.js`](src/lib/images.js) и вставьте свой Access Key от Unsplash —
   где его взять, написано там же в комментарии. Без ключа работает всё, кроме кнопки
   «Подобрать картинку»: картинку по-прежнему можно выбрать с диска.

5. **Запустите:**

   ```bash
   npm install
   npm run dev
   ```

   Приложение откроется на http://localhost:5173. Зарегистрируйтесь по email и паролю.

> Если в проекте Supabase включено подтверждение email (по умолчанию — да), после регистрации
> придёт письмо со ссылкой. Чтобы входить сразу, отключите `Confirm email` в
> `Authentication` → `Providers` → `Email`.

## Импорт готовой колоды

В [`data/cards_export.json`](data/cards_export.json) лежит выгрузка из Quizlet
(папка «Английский с нуля», 47 наборов, 1721 карточка). Залить её в свой аккаунт:

```bash
npm run import -- --dry-run   # разобрать файл и показать план, база не трогается
npm run import                # импорт
```

Данные закрыты политиками RLS, поэтому скрипт спросит email и пароль вашего
аккаунта — того самого, под которым вы заходите в приложение. Пароль нигде не
сохраняется; можно передать его через переменные `SUPABASE_EMAIL` и
`SUPABASE_PASSWORD`, если запускаете неинтерактивно.

Импорт идемпотентный: папка ищется по имени, набор — по имени внутри папки,
карточка — по позиции внутри набора. Второй запуск дублей не создаст и ничего не
удалит, а если импорт оборвётся на середине — просто запустите ещё раз, добавятся
недостающие карточки.

**Как работает Learn.** У каждой карточки есть `mastery_level` от 0 до 3 — это счётчик верных
ответов подряд, он живёт в базе и не сбрасывается между сессиями. Слово с уровнем 3 считается
выученным и из сессии выбывает. Ошибка обнуляет счётчик. Чем ниже уровень и чем больше ошибок
в этой сессии, тем чаще слово выпадает (вес `[6, 4, 2]` по уровню плюс `+3` за каждую ошибку).
Новые слова спрашиваются выбором из четырёх вариантов, знакомые — ручным вводом. Ответ
сверяется мягко: регистр, лишние пробелы, артикль и знаки препинания не важны, содержимое
скобок (`write (рАйт)`, `(a) mother`) можно не набирать, а варианты через `/` или запятую
засчитываются любым. Термины, которые нереально набрать без опечатки — многострочные диалоги
и длинные фразы, — спрашиваются только выбором: иначе слово никогда не дошло бы до уровня 3
и сессия не закончилась бы.

**«Учить все».** Кнопка на уровне папки запускает Flashcards или Learn сразу по всем словам
всех наборов. Это сборка на лету: новых записей в базе не появляется, прогресс пишется тем же
карточкам по их `id`. По умолчанию порядок как в папке (наборы по позиции, внутри — карточки
по позиции), опция «перемешать» выключена.

Безопасность: данные закрыты политиками RLS на стороне Supabase — каждый запрос видит только
строки со своим `auth.uid()`. Картинки лежат в бакете по пути `<user_id>/<файл>`, и писать
в чужую папку политики не дадут. Поэтому `anon`-ключ во фронтенде безопасен.
